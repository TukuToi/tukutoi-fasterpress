Speed Up your WordPress Website!

== Description ==

TukuToi FasterPress helps you speed up your WordPress websites easily.

It allows to remove (dequeue) styles and scripts with simple to use settings and also you can use it to "log" all loaded and registered styles and scripts.

You can disable WordPress Emojy scripts and also disable the Emojys even if devices would support them.

Easily exclude certain archives or single posts/pages from the optimization in case some scripts may be needed locally.


== Installation ==

1. Upload the Plugin to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Activate the "Log styles and scripts" settings in Dashboard > TukuToi > Speed 
1. Load the Front End of your website once
1. Deactivate the "Log styles and scripts" settings in Dashboard > TukuToi > Speed 
1. Start dequeing styles and scritps and use other optimization features of the plugin...
